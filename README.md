# protogo; 

## the library for data prototyping
It could be shown you data prototyping result easily.

## Protogo is simple.
```javascript
// init
var _protogo = new Protogo();
_protogo.init("data source");

// you can search some keyword which you find
_protogo.search("keyword");
```
